How the public demo works

A compact deployment designed for open access, transparent limitations and private processing.

01

Trained image model

A SigLIP Vision Transformer evaluates camera and generated-image traces from whole images, crops and sampled video frames.

02

Video analyst

Short videos are decoded locally, sampled across time and checked for frame disagreement, temporal jumps, freeze loops and unusual motion.

03

Correction memory

Reviewed exact and perceptual matches return faster, and new user corrections are stored locally for future runs.

04

Portable report

Download scores, frame summaries, web-search leads, limitations and feedback data for review or later retraining.

03

Useful evidence,
without theatrical certainty.

This public demo uses the Apache-2.0 SigLIP image detector `Ateeqq/ai-vs-human-image-detector`, exported correction memory and browser-only temporal video signals. Results can shift with new generators, edits, screenshots, compression and cross-origin video limits, so do not use them as the only basis for high-impact decisions.
